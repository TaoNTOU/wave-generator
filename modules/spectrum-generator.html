Pierson-Moskowitz Spectrum

Fully developed sea spectrum under steady wind conditions. 

S(f)=αg²(2π)^-4 f^-5 exp[-5/4(fp/f)^4]
Parameter Description
α Phillips constant
fp Peak frequency
f Frequency (Hz)
S(f) Spectral density (m²/Hz)

JONSWAP Spectrum

Fetch-limited growing sea spectrum with peak enhancement. 

S(f)=PM(f)·γ^exp[-(f-fp)²/(2σ²fp²)]
Parameter Description
γ Peak enhancement factor
σ Spectral width parameter
fp Peak frequency
S(f) Spectral density (m²/Hz)

Bretschneider Spectrum

Engineering two-parameter spectrum commonly used in coastal engineering. 

S(f)=5/16 Hs² fp⁴ f^-5 exp[-5/4(fp/f)^4]
Parameter Description
Hs Significant wave height
fp Peak frequency
f Frequency (Hz)
S(f) Spectral density (m²/Hz)

TMA Spectrum

Shallow water modified JONSWAP spectrum. 

S_TMA(f)=S_JONSWAP(f)·Φ(ω,h)
Parameter Description
Φ Shallow water correction factor
ω Angular frequency
h Water depth
S(f) Spectral density (m²/Hz)
